# Privacy Policy — Umbraco Tooltip Manager

_Last updated: 2026-07-14_

Umbraco Tooltip Manager ("the extension") is a UI utility that hides or toggles
tooltip/help text and adjusts small UI behaviours on Umbraco backoffice pages.

## What data the extension handles

- **Settings you choose** (tooltip mode, timeouts, CSS selectors, keyboard-shortcut
  toggles) are saved using Chrome's `storage.sync` API.
- **A list of website hostnames** where an Umbraco page was detected, so you can
  enable/disable the extension per site. Hostnames are added only for pages whose
  URL contains `umbraco`.

## Where the data goes

- All data is stored **locally in your browser** and, if you are signed in to
  Chrome, synced to your own Google account via Chrome Sync. This is handled by
  Chrome itself.
- The extension has **no backend server**. It does **not** send, sell, share, or
  transmit any data to the developer or any third party.
- The extension performs **no analytics or tracking**.

## Page access

The extension reads and modifies the page DOM **only on URLs that contain
`umbraco`** in order to hide/toggle tooltips and adjust panel/notification UI. It
does not read page content on any other site.

## Your control

You can view, edit, or delete all stored data from the extension's Settings page,
or by removing the extension, which clears its storage.
